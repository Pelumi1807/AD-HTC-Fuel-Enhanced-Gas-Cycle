## 📋 Project Overview

This project implements a thermodynamic analysis tool for an **Adsorption-Hydrothermal Carbonization (AD-HTC) Fuel-Enhanced Gas Power Cycle**. The system combines biomass upgrading (HTC) and biogas enhancement (AD) with a high-temperature gas turbine cycle to achieve superior thermal efficiency and fuel flexibility.
